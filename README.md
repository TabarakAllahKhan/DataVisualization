
# Data Visualization Using Python 

This project demonstrates data visualization techniques in Python, including bar charts, line plots, scatter plots, and interactive dashboards. It is designed to help data analysts and students understand data patterns quickly and present insights effectively.



## Tech Stack

+ Pandas
+ Plotly
+ Seaborn


## Installation

Install the libraries using `requirements.txt`

```bash
  pip install -r requirements.txt

```
